How It Works

Our deep learning pipeline analyses chest X-ray images through a trained Convolutional Neural Network to detect signs of pneumonia.

1. Upload X-Ray

Sign in and upload a chest X-ray image in PNG or JPEG format. The image is securely stored on our server for processing.

2. CNN Analysis

Our Convolutional Neural Network — trained on the Kaggle Chest X-Ray dataset — preprocesses and analyses the image in real time.

3. Instant Results

Receive a clear PNEUMONIA or NORMAL prediction along with a confidence score. All results are saved in your prediction history.

About the CNN Model

The model is a Sequential Convolutional Neural Network built with TensorFlow / Keras. It consists of three convolution blocks with MaxPooling layers followed by a fully connected Dense layer with Dropout for regularization.

  • 3 × Conv2D + MaxPooling2D layers
  • Dense layer (128 units) with ReLU activation
  • Dropout (0.5) to prevent overfitting
  • Sigmoid output for binary classification
  • Trained on Kaggle Chest X-Ray Pneumonia dataset
model = Sequential([
    Conv2D(32, (3,3), activation='relu', input_shape=(128,128,3)),
    MaxPooling2D(2,2),

    Conv2D(64, (3,3), activation='relu'),
    MaxPooling2D(2,2),

    Conv2D(128, (3,3), activation='relu'),
    MaxPooling2D(2,2),

    Flatten(),
    Dense(128, activation='relu'),
    Dropout(0.5),
    Dense(1, activation='sigmoid')
])
