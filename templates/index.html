Under the Hood

About the CNN Model

The model is a Sequential Convolutional Neural Network built with TensorFlow / Keras. It consists of three convolution blocks with MaxPooling layers followed by a fully connected Dense layer with Dropout for regularisation.

  • 3 × Conv2D + MaxPooling2D layers
  • Dense layer (128 units) with ReLU activation
  • Dropout (0.5) to prevent overfitting
  • Sigmoid output for binary classification
  • Trained on Kaggle Chest X-Ray Pneumonia dataset
model = Sequential([
    Conv2D(32, (3,3), activation='relu', input_shape=(128,128,3)),
    MaxPooling2D(2,2),

    Conv2D(64, (3,3), activation='relu'),
    MaxPooling2D(2,2),

    Conv2D(128, (3,3), activation='relu'),
    MaxPooling2D(2,2),

    Flatten(),
    Dense(128, activation='relu'),
    Dropout(0.5),
    Dense(1, activation='sigmoid')
])
